匿名者（英语：Anonymous）是一种理念，只要有相同想法或是理念任何人都可以加入。该组织发迹于2003年，随后在政治上形成一些共识，并发起自主性或自发性行为与活动。他们因其宣传及利用阻断服务攻击（DDoS）政府、宗教和企业网站而知名于世


匿名者于2003年起源于4chan的贴图区，贴图区能让许多在线和离线社区用户以一个无政府状态的方式交流，形成一个数字化的全球脑（digitized global brain）。匿名者的成员（也称为Anons）在示威或其他活动中一般带着盖伊·福克斯面具以隐藏身份。在匿名者发展早期，成员们会通常在在线社区协议一些不严格的自我约定目标，那些目标主要注重于娱乐，或弄一些恶作剧以达至幸灾乐祸的目的，但在2008年，匿名者开展了名为“Project Chanology”的行动，以抗议、恶作剧和黑客攻击的方式反对山达基教会，自此匿名者便开始合作参与更多与国际问题有关的黑客行动。
